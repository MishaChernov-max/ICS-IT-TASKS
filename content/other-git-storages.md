В качестве хранилища можно подключать не только GitLab, GitHub и Gitea. Gramax также поддерживает: Bitbucket, Gogs, GitFlic.

Чтобы подключить такое хранилище, на главной кликните *Загрузить существующий* и в вариантах хранилищ выберите *Git*. Затем заполните информацию о хранилище.

-  URL сервера Git -- указывается без уточнения компании, группы и репозитория.

-  Имя пользователя -- ваш username в Git-хранилище. Отображается в URL страницы профиля.

-  Токен -- Access-токен с правами на API и редактирование репозиториев.

   Если в сервисе нет возможности создать токен с доступом на редактирование репозиториев, отметьте пункт *Использовать пароль*. И укажите пароль от учетной записи вместо токена.

-  Имя автора и Почта -- ваши данные, которые будут отображаться в истории Git и в Gramax.

## Известное ограничение

В Gramax можно работать только с каталогами, которые уже существуют в хранилище.

Если каталог создан и существует только локально в Gramax, опубликовать его в такое хранилище не получится. Чтобы не потерять данные, есть обходные пути:

1. **Каталог создан в браузерной версии**: опубликуйте его в GitLab, GitHub или Gitea, затем перенесите его копию в свое Git-хранилище.

2. **Каталог создан в десктопной версии**: опубликуйте каталог с помощью консоли или любого Git-клиента. Затем удалите локальную версию каталога из Gramax и загрузите из хранилища.
