Чтобы с каталогами было удобно работать, их можно сгруппировать на главной странице по темам или отделам.

-  Группировка каталогов настраивается в файле `workspace.yaml`.

-  Настройки из файла `workspace.yaml` отображаются только у конкретного пользователя на конкретном устройстве. Но ими можно [поделиться с коллегами](./settings#%D0%BF%D0%BE%D0%B4%D0%B5%D0%BB%D0%B8%D1%82%D1%8C%D1%81%D1%8F-%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B0%D0%BC%D0%B8-%D1%81-%D0%BA%D0%BE%D0%BB%D0%BB%D0%B5%D0%B3%D0%B0%D0%BC%D0%B8) или [опубликовать на портал для чтения](./settings#%D0%BE%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%BE%D0%B2%D0%B0%D1%82%D1%8C-%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B8-%D0%BD%D0%B0-%D0%BF%D0%BE%D1%80%D1%82%D0%B0%D0%BB-%D0%B4%D0%BE%D0%BA%D1%83%D0%BC%D0%B5%D0%BD%D1%82%D0%B0%D1%86%D0%B8%D0%B8).

-  По умолчанию во всех пространствах устанавливается группировка «Папки».

-  Пользователь может настроить собственную группировку, добавив часто используемые каталоги в группу «Избранное» -- она отображается над всеми другими секциями и группами.

## Настроить группировку

Тип группировки каталогов задается в файле `workspace.yaml` для каждой группы отдельно:

-  `view: section` -- секции. Секция может быть только на первом уровне.

-  `view: folder` -- папки. Папки могут вкладываться в друг друга и в секции без ограничений.

Для каждой секции необходимо указать код секции, название, а также каталоги, которые в нее входят. Пример конфигурации:

```yaml
name: Demo # Наименование пространства
icon: tag. # Иконка пространства
sections: 
  products: # Код секции 1, будет показываться в адресной строке при переходе по секциям
    title: Продукты # Название секции
    view: section # Отображение группировки каталогов (секция). Section - работает только для групп первого уровня
    catalogs: # Каталоги в секции
      - ad # Название репозитория
      - ac # Название репозитория
    sections: # Папка внутри секции
      security: # Код папки, будет показываться в адресной строке при переходе по папкам
        title: Информационная безопасность # Название папки
		view: folder # Отображение группировки каталогов (папка)
        icon: shield # Иконка папки
        catalogs: # Каталоги внутри папки
          - adr # Название репозитория
          - ap # Название репозитория
  company: # Код секции 2
    title: Внутренняя информация
  projects: # Код секции 3
    title: Проекты
```
