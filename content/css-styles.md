Вы можете изменить внешний вид приложения и портала документации. Например, чтобы он соответствовал корпоративным требованиям.

-  Стили задаются в приложении.

-  Стили применяются только в конкретном приложении (браузерном или десктопном) на конкретном устройстве.

-  Стилями можно поделиться с коллегами, а также [опубликовать их на портал документации](./../doc-portal/deploy-with-docker/_index).

Чтобы задать стили, перейдите в [настройки пространства](./settings) и кликните «CSS-стили».

## Переменные

:::lab:true Пример стилей

```css
#custom-style {
    --bg-main: rgba(243, 237, 255, 0.5);

    --accent-60: rgba(60, 32, 101, 0.51);
    --accent-80: rgba(76, 41, 130);
    --accent-100: rgb(125, 72, 192);
    --accent-120: rgb(164, 94, 255);
}

#custom-style[data-theme="light"] {
    /* Цвет ссылок в приложении */
    --color-link: #6E23C9;

    /* Сайдбар слева (навигация) */
    --color-nav-menu-bg: var(--bg-main);

    /* Сайдбар справа */
    --color-right-nav-bg: var(--bg-main);

    /* В левой навигации, снизу, панель с гитом */
    --version-control-primary: var(--accent-100);

    /* Статья */
    /* Так же эти переменные используются в компонентах статьи и модальных окнах */
    --color-article-heading-text: black;
    --color-article-text: black;

    /* Стандартная */
    --color-btn-default-bg: var(--accent-100);
    --color-btn-default-text: #fff;
    --border-btn-default: 1px solid transparent;
    --opacity-btn-default-bg-hover: 1;
    --color-btn-default-text-hover: #fff;
    --color-btn-default-bg-hover: var(--accent-120);

    /* Подчеркнутая снизу */
    --color-btn-underline-text-hover: var(--accent-100);
}

#custom-style[data-theme="light"] {
    /* Цвет фона главной страницы */
    --color-home-bg: var(--bg-main);

    /* Цвет границы карточки */
    --color-home-card-border: rgba(125, 68, 214, 0.1);
    --color-home-card-border-hover: rgba(125, 68, 214, 0.15);

    /* Цвет заднего фона карточки */
    --color-home-card-bg-hover: rgba(125, 68, 214, 0.05);

    /* Тень карточки при наведении */
    --box-shadow-home-card: 0px 0px 12px -5px #00000040;
}
```

:::

### **Главная страница**

-  Цвет текста

   -  `--color-home-card-link` -- цвет текста карточек.

   -  `--color-home-card-link-hover` -- цвет текста карточек при наведении.

-  Цвет фона и границ

   -  `--color-home-bg` -- цвет общего фона.

   -  `--color-home-card-bg` -- цвет фона карточек.

   -  `--color-home-card-bg-hover` -- фон карточек при наведении.

   -  `--color-home-card-border` -- цвет границы у карточек .

   -  `--color-home-card-border-hover` -- цвет границы карточек при наведении.

-  Тени

   -  `--box-shadow-home-card` -- тень для карточек.

### Цвета

-  Цвет текста в приложении

   -  `--color-primary-general` -- цвет основного текста в приложении.

   -  `--color-primary` -- цвет подчеркнутого текста в приложении.

-  Цвет иконок

   -  `--color-primary-general` -- цвет иконок в приложении.

      *Наследует цвет основного текста в приложении*

-  Цвет ссылок

   -  `--color-link` -- цвет ссылок в приложении.

-  Фон подсказок

   -  `--color-tooltip-background` -- фон всплывающих подсказок.

### Боковые панели

В приложении в открытом каталоге.

-  Цвет текста левого меню

   -  `--color-nav-item` -- цвет текста пунктов меню.

   -  `--color-nav-item-selected` -- цвет текста активного пункта меню.

-  Цвет фона

   -  `--color-nav-menu-bg` -- фон левой панели.

   -  `--color-right-nav-bg` -- фон правой панели.

### Строка состояния репозитория

В приложении в левом нижнем углу каталога.

-  **Цвета:**

   -  `--version-control-primary` – основной цвет для элементов управления версиями.

### Статья и модальные окна

Основное пространство с текстом в открытом каталоге. А также все всплывающие окна.

-  Текст

   -  `--color-article-heading-text` -- цвет заголовков.

   -  `--color-article-text` -- цвет основного текста.

### Примечания

Блок оформления текста в статье с типом *Цитата*.

-  Цвета

   -  `--color-admonition-quote-border` -- цвет границы примечаний.

   -  `--color-admonition-quote-bg` -- фон примечаний.

   -  `--color-admonition-text` -- текст внутри примечаний.

### Кнопки

-  Обычная кнопка

   -  `--color-btn-default-bg` -- цвет фона.

   -  `--color-btn-default-text` -- цвет текста.

   -  `--border-btn-default` -- цвет границы.

   -  `--opacity-btn-default-bg-hover` -- непрозрачность при наведении.

   -  `--color-btn-default-text-hover` -- цвет текста при наведении.

   -  `--color-btn-default-bg-hover` -- цвет фона при наведении.

-  Подчеркнутая кнопка

   -  `--color-btn-underline-text` -- цвет текста.

   -  `--font-weight-btn-underline` -- толщина шрифта.

   -  `--padding-btn-underline` -- внутренние отступы.

   -  `--margin-btn-underline` -- внешние отступы.

   -  `--color-btn-underline-text-hover` -- цвет текста при наведении.

   -  `--color-btn-underline-bg-hover` -- фон при наведении.

   -  `--text-decoration-btn-underline-hover` -- подчеркивание при наведении.

:::info Светлая и темная темы

По умолчанию стили задаются для светлой темы. Для того что бы указать стили для темной темы, нужно поменять значение селектора `data-theme` с `light` на `dark`.

:::

## Шрифт

Вы можете изменить шрифт во всем приложении. Для этого нужно подключить в css-шрифт, после этого указать его в переменной `--app-font-famaly`.

-  Шрифт можно подобрать на сайте <https://fonts.google.com/>.

-  Для корректной работы шрифта в редакторе, нужно добавлять шрифт, имеющий начертания:

   -  Italic.

   -  Разную толщину шрифта: 300/400/500/600.

```css
@import url('https://fonts.googleapis.com/css2?family=Montserrat:ital,wght@0,100..900;1,100..900&display=swap');

#custom-style {
	--app-font-famaly: 'Montserrat', sans-serif;
}
```

## Логотип главной страницы

Вы можете задать логотип, который будет отображаться в левом верхнем углу на главной странице. Для этого также кликните название пространства на главной странице, а затем нажмите [icon:pencil-line].

-  Логотип можно добавить как для темной, так и для светлой темы.

-  Если логотип добавлен только для светлой темы, он будет также использоваться для темной.

-  Логотип должен быть в формате SVG.

-  Максимальный размер -- 500 КБ.

-  Высота логотипа по умолчанию ограничена 40px, изменить эту переменную или задать другую можно с помощью стилей.

## Логотип каталога

Логотип каталога хранится в репозитории и [задается в системном файле](./../catalog/settings/logo).

По умолчанию он отображается в правом верхнем углу карточки каталога. Но расположение логотипа можно изменить.

1. Для одного каталога: в стилях укажите каталог -- `[data-catalog-card="имя-каталога"]`. Затем измените параметры отображения логотипа.

   ```css
   [data-catalog-card="gramax"] .catalog-title-logo {
   	height: 5.5rem;
   	top: 0.5rem;
   	right: 0;
   }
   ```

2. Для всех каталогов: без указания каталога измените параметры отображения логотипа.

   ```css
   .catalog-title-logo {
   	height: 5.5rem;
       top: 0.5rem;
       right: 0;
   }
   ```
