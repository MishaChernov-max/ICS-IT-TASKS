---

Развернуть портал для читателей можно на собственном сервере с помощью Docker-образа Gramax. Поддерживается развёртывание в Docker от версии 20.10.

:::tip 

Если у вас нет собственного сервера, портал для читателей можно также [развернуть в Yandex Cloud](./deploy-on-yandex-cloud).

:::

:::info Развернуть с помощью Podman

Если вы разворачиваете портал с помощью Podman:

1. Создайте папку, в которой будут находиться все каталоги. Это можно сделать по пути, который вы указали в переменной среде параметром `ROOT_PATH`. Если не указали -- создайте папку `gramax` рядом с файлом `docportal-docker-compose.yaml`.

2. Замените во всех командах `docker` на `podman`.

:::
