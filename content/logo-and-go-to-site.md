---

В каждом каталоге в левом верхнем углу по умолчанию находится кнопка *На главную*. Вы можете заменить ее на иконку компании, а также настроить переход на сторонний ресурс -- например, на сайт вашей компании.

:::tip 

Это полезно в случае, если на портале всего один каталог и читателю не нужно выходить на главную.

:::

[image:./logo-and-go-to-site.png:::0,0,100,100::annotation,2.8549,27.5,Кнопка перехода ,top-left:2880px:267px:center]

Для этого при [разворачивании](./deploy-with-docker/_index) портала документации или при [сборке](./_index) статического сайта укажите переменные:

-  `LOGO_IMAGE_URL` -- URL изображения, которое будет отображаться вместо кнопки *На главную*.

-  `LOGO_LINK_URL` -- ссылка, куда будет переходить читатель при клике на иконку. Если URL не указан, ведет в корень домена.

-  `LOGO_LINK_TITLE` -- тултип, который будет показываться при наведении курсора на иконку. Если не указан, выводится *Перейти на LOGO_LINK_URL*.

:::lab 

Параметры `LOGO_LINK_URL` и `LOGO_LINK_TITLE` работают только при наличии `LOGO_IMAGE_URL`.

:::
